## Support Vector Machine Example
SVM Algorithm example about given wine dataset.

#### Used Python Version
- 3.7.2

#### Requirements

- pandas
- numpy
- matplotlib
- sklearn

#### Install Requirements
`pip install -r requirements.txt`

#### Run Script
`python svm.py`
